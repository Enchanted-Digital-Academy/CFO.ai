# CFO.ai

CFO.ai is an AI-powered financial management dashboard designed to help Small and Medium-sized Enterprises (SMEs) understand the health of their business beyond account balances.

## Tech Stack

- React
- TypeScript
- Vite
- ESLint

## Getting Started

```bash
npm install
npm run dev
```

## Branching Strategy

- main
- develop
- feature/dev-1-auth-layout
